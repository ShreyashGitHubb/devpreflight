# Homebrew DevPreflight

This is the Homebrew tap for [DevPreflight](https://github.com/devpreflight/devpreflight), a CLI tool for validating development configurations and best practices.

## Installation

```bash
# Add the tap
brew tap devpreflight/devpreflight

# Install devpreflight
brew install devpreflight
```

## Updating

```bash
brew upgrade devpreflight
```

## Development

This tap is automatically updated by GoReleaser when a new version of DevPreflight is released.
